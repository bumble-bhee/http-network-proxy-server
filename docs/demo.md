# Demo Instructions – HTTP Proxy Server

This document describes how to build, run, and test the HTTP proxy server.

⚠️ IMPORTANT NOTE

All commands mentioned in this document must be executed from the **project root directory**.

Do NOT run commands from inside `docs/`, `config/`, or `tests/` folders.

---

From the project root directory:
